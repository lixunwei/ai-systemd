## 二、源码索引结构

索引文件位于 `.ai-search/` 目录下：

```
.ai-search/
├── zoekt/          # zoekt 全文索引（113MB）
├── ctags/
│   └── tags        # Universal Ctags 符号索引（61462 条）
└── cscope/
    ├── cscope.out  # cscope 交叉引用数据库（17MB）
    ├── cscope.po.out  # 反向索引（25MB）
    └── cscope.files   # 源文件列表（2157 个文件）
```

### 索引使用方法

```bash
# 全文搜索
/home/nio/lixunwei/codex/.tools/zoekt/bin/zoekt \
  -index_dir .ai-search/zoekt -r '搜索模式'

# 符号查找
readtags -t .ai-search/ctags/tags -en '符号名'

# 函数调用者查找
cscope -d -f .ai-search/cscope/cscope.out -L3 '函数名'

# 函数被调用查找
cscope -d -f .ai-search/cscope/cscope.out -L2 '函数名'

# 符号引用查找
cscope -d -f .ai-search/cscope/cscope.out -L0 '符号名'
```
